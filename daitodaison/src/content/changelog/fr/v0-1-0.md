Nous avons enrichi le kit UI avec de nouveaux composants essentiels pour construire plus vite.

### Nouveaux composants
- **Timeline** : Visualisez l'historique de votre entreprise ou votre feuille de route.
- **Logo Cloud** : Affichez professionnellement les logos de partenaires avec un effet survol niveaux de gris vers couleur.
- **Breadcrumbs** : Navigation améliorée pour les pages imbriquées.

Également inclus :
- Mise à jour du `Header` avec positionnement sticky.
- Amélioration de l'accessibilité (ARIA labels) sur tous les boutons.
