Une mise à jour critique axée sur les dépendances de sécurité et l'optimisation de la compilation.

### 🛡️ Mises à Jour de Sécurité
- Mise à niveau de toutes les dépendances pour corriger les vulnérabilités connues.
- Implémentation des en-têtes Content Security Policy (CSP) dans la configuration par défaut.
- Assainissement des champs de saisie dans le formulaire de contact pour prévenir les attaques XSS.

### 🚀 Performance
- **Optimisation des Images** : Configuration améliorée d'`astro:assets` pour des fichiers WebP 20 % plus légers.
- **Chargement Différé** : Chargement natif différé activé pour toutes les images sous le pli.

> [!IMPORTANT]
> Cette mise à jour est recommandée pour tous les déploiements en production.
