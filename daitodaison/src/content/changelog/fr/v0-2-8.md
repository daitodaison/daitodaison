Nous avons audité l'intégralité du code pour atteindre des scores Lighthouse parfaits.

### ⚡ Performance
- **Intégration Partytown** : Exécution hors fil principal pour les scripts tiers.
- **Optimisation des Polices** : Polices auto-hébergées avec `font-display: swap` activé par défaut.
- **CSS Critique** : Styles critiques en ligne pour un Premier Rendu de Contenu (FCP) instantané.

### Composants
- **Image** : Nouveau composant wrapper `Picture` pour la direction artistique avancée.
