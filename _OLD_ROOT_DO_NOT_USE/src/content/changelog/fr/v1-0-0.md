Nous finalisons la feuille de route pour notre première version majeure. L'accent est mis sur trois piliers clés :

### 🌍 Internationalisation (i18n)
Nous construisons un système i18n robuste à partir de zéro.
- **Support multilingue** : Routage transparent pour `en`, `fr`, `de` et `ar`.
- **Support RTL** : Prise en charge de premier ordre pour les langues de droite à gauche comme l'arabe.
- **Traductions typées** : Un dictionnaire typé pour tous les composants de l'interface utilisateur.

### 🔍 Capacités de recherche
Intégration d'un puissant moteur de recherche statique pour des résultats instantanés.
- **Intégration Pagefind** : Indexation de recherche statique sans configuration.
- **Palette de commande** : Interface `Cmd+K` pour une navigation rapide.
- **Indexation multilingue** : Les résultats de recherche respectent la langue actuelle.

### 🔓 Open Source
Préparation du dépôt pour la diffusion publique.
- **Licence MIT** : Entièrement open source pour l'utilisation par la communauté.
- **Guide de contribution** : Directives claires pour les contributeurs.
- **Dépôt public** : Passage du développement privé au dépôt GitHub public.
