Cette mise à jour se concentre sur l'élimination des bugs et le raffinement de l'expérience utilisateur.

### 🐛 Corrections de bugs
- Correction d'un décalage de mise en page dans le composant `Hero` sur Safari.
- Résolution des conflits z-index entre la `Navbar` et les superpositions `Modal`.
- Correction des ratios de contraste pour le texte en Mode Sombre afin de respecter les normes WCAG AA.

### Améliorations
- **Typographie** : Ajustement des hauteurs de ligne pour une meilleure lisibilité sur les contenus longs.
- **Pied de page** : Espacement affiné et ajout des états de survol pour les icônes de réseaux sociaux.
