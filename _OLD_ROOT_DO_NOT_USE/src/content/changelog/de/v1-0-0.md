Wir finalisieren die Roadmap für unser erstes großes Release. Der Fokus liegt auf drei wichtigen Säulen:

### 🌍 Internationalisierung (i18n)
Wir bauen ein robustes i18n-System von Grund auf neu.
- **Mehrsprachiger Support**: Nahtloses Routing für `en`, `fr`, `de` und `ar`.
- **RTL-Support**: Erstklassige Unterstützung für Sprachen, die von rechts nach links geschrieben werden, wie Arabisch.
- **Typsichere Übersetzungen**: Ein typsicheres Wörterbuch für alle UI-Komponenten.

### 🔍 Suchfunktionen
Integration einer leistungsstarken statischen Suchmaschine für sofortige Ergebnisse.
- **Pagefind-Integration**: Statische Suchindizierung ohne Konfiguration.
- **Befehlspalette**: `Cmd+K`-Schnittstelle für schnelle Navigation.
- **Mehrsprachige Indizierung**: Suchergebnisse berücksichtigen die aktuelle Sprache.

### 🔓 Open Source
Vorbereitung des Repositories für die öffentliche Freigabe.
- **MIT-Lizenz**: Vollständig Open Source für die Nutzung durch die Community.
- **Beitragsleitfaden**: Klare Richtlinien für Mitwirkende.
- **Öffentliches Repository**: Übergang von privater Entwicklung zu öffentlichem GitHub-Repo.
